# Korkmaz Elektrik - Professional Electrical Services Website

A modern, high-performance website for an electrical services company built with cutting-edge web technologies. Korkmaz Elektrik showcases professional electrical services, project portfolios, and client resources through an intuitive, responsive platform that delivers exceptional user experience across all devices.

---

## Project Overview

Korkmaz Elektrik is a modern, professional website designed to showcase electrical services and projects. The platform provides a user-friendly interface for customers to explore services, view completed projects, and connect with the company. The website combines clean design with intuitive navigation to create an engaging experience for potential clients. Built using Next.js 15 with App Router architecture, the project implements static site generation for optimal performance and SEO.

## Key Features & Achievements

### Service Showcase

- **Service Presentation**: Comprehensive display of electrical services with detailed descriptions and interactive hover effects

- **Project Portfolio**: Visual showcase of completed electrical projects with dynamic routing (`/projects/[slug]`) and static generation

- **Service Categories**: Well-organized service categories for easy navigation with dynamic image positioning on hover

- **Visual Content**: High-quality images optimized with Next.js Image component and responsive gallery layouts

### User Experience

- **Intuitive Navigation**: Clean and simple navigation structure with client-side routing and smooth transitions

- **Responsive Design**: Fully responsive website using Tailwind CSS 4 with mobile-first approach, working seamlessly across all devices

- **Contact Integration**: Contact forms integrated with FormSubmit API for reliable email delivery without backend dependencies

- **Professional Presentation**: Modern design with dark/light theme support using next-themes v0.4.6

### Content Management

- **Markdown-Based Content**: MDX files for blog posts and project descriptions using gray-matter v4.0.3 for front matter parsing

- **Static Content Generation**: Pre-rendered pages at build time using `generateStaticParams()` for optimal performance

- **Dynamic Metadata**: Server-side metadata generation with `generateMetadata()` for SEO optimization

- **Content Processing**: Markdown to HTML conversion using remark v15.0.1 and remark-html v16.0.1

### Technical Excellence

- **Performance Optimization**: Static site generation with `output: 'export'` configuration for CDN deployment

- **SEO Friendly**: Comprehensive meta tags, dynamic metadata generation, and semantic HTML structure

- **Accessibility**: Accessible design following WCAG standards with Radix UI components (v1.2.3)

- **Mobile-First Approach**: Responsive design implemented with Tailwind CSS 4 and breakpoint-based layouts

## Technical Highlights

### Performance Optimization

- **Static Site Generation**: Full static export using `output: 'export'` in next.config.ts for CDN deployment

- **Image Optimization**: Next.js Image component with quality settings (25, 50, 75, 100) and responsive sizing

- **Code Splitting**: Automatic route-based code splitting with Next.js App Router

- **Font Optimization**: Google Fonts (Manrope) optimization using next/font/google with subset loading

- **Lazy Loading**: Component-level lazy loading with React.lazy and dynamic imports

- **Build Optimization**: Optimized production builds with tree shaking, minification, and asset optimization

### Architecture & Implementation

- **App Router Architecture**: Next.js 15 App Router with Server Components and Client Components separation

- **File-Based Routing**: Dynamic routes using `[slug]` pattern for projects and blog posts

- **Server Components**: Server-side rendering for metadata generation and static content

- **Client Components**: Interactive components marked with "use client" directive for client-side interactivity

- **TypeScript Configuration**: Strict TypeScript with ES2017 target, ES modules, and path aliases (@/*)

- **Module Resolution**: Bundler-based module resolution for optimal build performance

### Content Management System

- **Markdown Processing**: Server-side Markdown processing using gray-matter and remark

- **Front Matter Parsing**: YAML front matter extraction for metadata (title, slug, description, images)

- **Static Path Generation**: `generateStaticParams()` for pre-rendering all dynamic routes at build time

- **Content Filtering**: Field-based content filtering for optimized data fetching

- **Image Processing**: Automatic image path processing and HTML conversion in markdown content

### User Interface Implementation

- **Component Architecture**: Modular component structure with reusable UI components

- **Theme System**: Dark/light mode with ThemeProvider and system preference detection

- **Responsive Design**: Mobile-first approach with Tailwind breakpoints (sm, md, lg, xl, 2xl)

- **Interactive Elements**: Hover effects, transitions, and animations using CSS and React hooks

- **Form Handling**: Client-side form state management with React hooks and async submission

### SEO & Metadata

- **Dynamic Metadata**: Server-side metadata generation with `generateMetadata()` function

- **Structured Data**: Semantic HTML5 elements for better search engine understanding

- **Meta Tags**: Comprehensive meta tags including title, description, and Open Graph tags

- **Robots Configuration**: Configurable robots meta tags for search engine indexing control

- **Sitemap Ready**: Static generation enables easy sitemap creation

### Build & Deployment Configuration

- **Static Export**: Full static site generation with `output: 'export'` for CDN hosting

- **Trailing Slash**: Consistent URL structure with trailing slash configuration

- **Custom Dist Directory**: Build output to 'dist' directory for deployment flexibility

- **Image Configuration**: Unoptimized images for static export compatibility

- **ESLint Configuration**: Build-time ESLint with ignoreDuringBuilds option
